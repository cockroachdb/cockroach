# chatdemo — RAG Chat with CockroachDB Vectorizer

A terminal chat application that performs semantic search over documents stored
in CockroachDB using the vectorizer feature, with responses generated by a
local LLM via Ollama.

## Architecture

```
User ─── Terminal Chat ─── Go App ─── CockroachDB (embed + vector search)
                                  └── Ollama (LLM response generation)
```

The app sends each user question to CockroachDB, which embeds the query using
`embed()` and finds the most relevant document chunks via cosine distance with
a vector index — all in a single SQL round-trip. The retrieved context is then
sent to Ollama's local LLM to generate a grounded answer.

## Prerequisites

1. **CockroachDB** running with the embedding engine enabled:
   ```bash
   cockroach start-single-node --insecure \
     --embedding-libs=$HOME/onnxruntime/lib/libonnxruntime.dylib
   ```

2. **Ollama** running with a model pulled:
   ```bash
   ollama pull llama3.1
   ollama serve  # if not already running
   ```

3. **Go 1.22+**

## Usage

### 1. Set up the demo dataset

```bash
cd demos/chatdemo
go run . --setup
```

This creates a `documents` table with 10 sample articles, and a vectorizer that
auto-embeds them in the background.

### 2. Wait for embeddings

The vectorizer runs every minute. Check progress:

```sql
SELECT count(*) FROM documents_embeddings;
```

Once this returns rows (may take 1-2 minutes), you're ready to chat.

### 3. Start chatting

```bash
go run .
```

Try queries like:
- "What is the security policy for passwords?"
- "How do backups work?"
- "What is the incident response process for critical outages?"
- "How should I optimize query performance?"

### Flags

| Flag | Default | Description |
|------|---------|-------------|
| `--db` | `postgresql://root@localhost:26257/defaultdb?sslmode=disable` | CockroachDB URL |
| `--ollama` | `http://localhost:11434` | Ollama API URL |
| `--model` | `llama3.1` | Ollama model name |
| `--top-k` | `5` | Number of chunks to retrieve |
| `--setup` | `false` | Create demo dataset and exit |
