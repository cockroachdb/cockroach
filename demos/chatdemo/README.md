# chatdemo — RAG Chat with CockroachDB Vectorizer

A terminal chat application that performs semantic search over CS research
papers from [arXiv](https://arxiv.org/archive/cs) stored in CockroachDB
using the vectorizer feature, with responses generated by a local LLM via
Ollama.

## Architecture

```
User ─── Terminal Chat ─── Go App ─── CockroachDB (embed + vector search)
                                  └── Ollama (LLM response generation)
```

The app sends each user question to CockroachDB, which embeds the query using
`embed()` and finds the most relevant paper chunks via cosine distance with
a vector index — all in a single SQL round-trip. The retrieved context is then
sent to Ollama's local LLM to generate a grounded answer.

## Prerequisites

1. **CockroachDB** running with the embedding engine enabled:
   ```bash
   cockroach start-single-node --insecure \
     --embedding-libs=$HOME/onnxruntime/lib/libonnxruntime.dylib
   ```

2. **Ollama** running with a model pulled:
   ```bash
   ollama pull llama3.1
   ollama serve  # if not already running
   ```

3. **Go 1.22+**

4. **Internet access** (for fetching papers from arXiv during setup)

## Usage

### 1. Set up the demo dataset

```bash
cd demos/chatdemo
go run . --setup
```

This fetches ~10,000 CS research papers from arXiv across 10 categories
(cs.AI, cs.CL, cs.CR, cs.CV, cs.DB, cs.DC, cs.DS, cs.LG, cs.SE, cs.PL),
loads them into a `papers` table, and creates a vectorizer that auto-embeds
them in the background.

The setup fetches papers in batches and respects arXiv API rate limits, so
it takes several minutes to complete.

### 2. Wait for embeddings

The vectorizer runs every minute. Check progress:

```sql
SELECT count(*) FROM papers_embeddings;
```

Embedding ~10,000 papers may take a while depending on your hardware.

### 3. Start chatting

```bash
go run .
```

Try queries like:
- "What papers are there on transformer architectures?"
- "Find database papers on query optimization"
- "Recent machine learning papers on reinforcement learning from 2024"
- "What research has been done on distributed consensus?"
- "Show me cryptography papers about zero-knowledge proofs"

### Flags

| Flag | Default | Description |
|------|---------|-------------|
| `--db` | `postgresql://root@localhost:26257/defaultdb?sslmode=disable` | CockroachDB URL |
| `--ollama` | `http://localhost:11434` | Ollama API URL |
| `--model` | `llama3.1` | Ollama model name |
| `--top-k` | `5` | Number of chunks to retrieve |
| `--setup` | `false` | Fetch papers from arXiv and set up dataset |

## Table Schema

```sql
CREATE TABLE papers (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    arxiv_id STRING NOT NULL UNIQUE,
    title STRING NOT NULL,
    authors STRING NOT NULL,
    abstract TEXT NOT NULL,
    category STRING NOT NULL,
    pdf_link STRING NOT NULL,
    published TIMESTAMP NOT NULL
);
```

The vectorizer auto-generates a `papers_embeddings` companion table with
chunked embeddings of each paper's title and abstract.
